# C++ TCP 多人聊天室（服务器 + 命令行客户端）

一个基于 C++ 的简单多人聊天室项目，包括：

- **服务器端**：运行在 Linux 上，使用阻塞式 TCP socket，每个客户端一个线程。
- **命令行客户端**：在用户自己的 Linux 电脑上运行，通过 IP + 端口连接聊天室服务器进行聊天。

适合作为「Linux + C++ 网络编程入门」的小型实战项目。

---

## 功能特性

- 使用 TCP 协议，支持多个客户端同时在线聊天
- 简单文本行协议：
  - 客户端连接后第一行发送昵称
  - 后续每一行都是聊天消息
- 服务器广播：
  - 将某个客户端的消息转发给所有在线客户端
  - 消息格式：`[昵称] 消息内容`
- 客户端双线程模型：
  - 一个线程负责读取键盘输入并发送消息
  - 另一个线程负责接收服务器广播并打印到终端
- 实现简单，但结构清晰，便于学习和扩展

---

## 技术栈与设计

### 服务器端

- 语言：C++（建议 C++17 及以上）
- 系统：Linux（开发与测试基于 Ubuntu 22.04）
- 网络：
  - 阻塞式 TCP socket
  - `socket / bind / listen / accept / recv / send`
- 并发模型：
  - 主线程执行 `listen + accept`
  - 每接受一个新连接，就创建一个客户端线程
  - 使用 `std::thread` 实现
- 线程同步：
  - 使用 `std::mutex` + `std::lock_guard` 保护「客户端列表」等共享数据

### 客户端

- 阻塞式 TCP socket
- 双线程：
  - 发送线程：循环读取 `std::cin`，输入 `/quit` 退出
  - 接收线程：循环从 socket 读取一行消息并打印
